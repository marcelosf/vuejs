## Instalação

```
npm install -g webpack-dev-server

npm install -g babel

npm install -g node-sass

npm install -g webpack

npm install --no-bin-link
```

## Comandos

| Comando     | Descrição                   |
|:-----------:|:---------------------------:|
|npm start    |Inicializa webpack-dev-server|
|npm run build| roda webpack                |
